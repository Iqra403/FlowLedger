# 💰 FlowLedger (TreasuryFlow)

A full-stack **Finance & Cash Flow Management System** built with:

- Spring Boot (Backend)
- React + Vite (Frontend)
- MySQL Database
- JWT Authentication
- REST APIs

---

## 🚀 Features

### 👤 Authentication
- User Registration
- Login with JWT Security
- Protected Routes

### 🏦 Bank Accounts
- Add Bank Account
- View all accounts
- Track current balance

### 💸 Transactions
- Add Income / Expense
- Auto update bank balance
- Transaction history

### 📊 Dashboard
- Total Cash Overview
- Total Income & Expense
- Recent Transactions
- Cash Flow Forecast (Projection)
